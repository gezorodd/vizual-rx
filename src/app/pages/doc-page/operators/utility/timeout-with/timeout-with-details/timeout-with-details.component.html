Deprecation Notes: Replaced with timeout. Instead of timeoutWith(100, a$, scheduler), use timeout with the configuration object: timeout({ each: 100, with: () => a$, scheduler }). Instead of timeoutWith(someDate, a$, scheduler), use timeout with the configuration object: timeout({first: someDate, with: () => a$, scheduler }). Will be removed in v8.

When the passed timespan elapses before the source emits any given value, it will unsubscribe from the source, and switch the subscription to another observable.
